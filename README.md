# Resume Parser AI

## Overview
Resume Parser AI is a machine learning project that classifies resumes into relevant job categories using Artificial Neural Networks (ANN), TF-IDF Vectorization and Principal Component Analysis (PCA).

## Features
- Resume text preprocessing
- TF-IDF feature extraction
- PCA dimensionality reduction
- ANN based classification
- Performance evaluation using confusion matrix

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- Matplotlib
- Seaborn

## Project Workflow
1. Data Cleaning
2. TF-IDF Vectorization
3. PCA Transformation
4. ANN Training
5. Resume Classification
6. Performance Evaluation
